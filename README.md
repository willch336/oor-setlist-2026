# ONE OK ROCK · 2026 Asia Tour Setlist

一頁式歌單，把演唱會 setlist 配上歌詞、副歌標記、日文歌的羅馬拼音，跟一鍵跳轉的串流連結整合在一起，方便看現場前複習、現場跟唱、現場後重聽。

## 怎麼用

直接打開：

**<https://willch336.github.io/oor-setlist-2026/oor-setlist.html>**

點任何一首歌，會看到：

- 完整歌詞（日文 + 英文）
- **副歌 / 重複段落**：黃字標記，方便抓主旋律
- **羅馬拼音**：日文歌每一行下方有 Hepburn-style romaji（灰色斜體），跟著唱日文用
- 5 個串流連結：Genius（英譯與註解）、YouTube（MV / Live）、Spotify、Apple Music，日文歌另外有 Lyrical Nonsense 的 Romaji

點左上角 **‹ Setlist** 可以隨時返回主歌單。

## iOS / Android 加到主畫面當 App

**iPhone（Safari）**

1. 打開上面那個網址
2. 點下方 **分享** 鍵
3. 選 **加入主畫面**
4. 取名 `OOR Setlist` → 加入

桌面會多一個 icon，點下去全螢幕、像原生 App。

**Android（Chrome）**

1. 打開網址
2. 右上 ⋮ 選單 → **加到主畫面**

## 顏色 / 標記說明

| 樣式 | 意思 |
|---|---|
| 黃字粗體 | 副歌、重複段、招牌句 |
| 灰色斜體 | 日文行的羅馬拼音（Hepburn） |
| `★` 圖示 | Bonus / 紀念曲 |
